# Count Uppercase and Lowercase Letters

## 📌 Project Overview
This is a simple Python program that counts the number of uppercase and lowercase letters in a user-entered string.

## 🎯 Problem Statement
Write a Python program to count the uppercase and lowercase letters in a given string.

## ✨ Features
- Accepts user input
- Counts uppercase letters
- Counts lowercase letters
- Displays the results clearly

## 🛠️ Technologies Used
- Python 3

## 📚 Concepts Used
- Strings
- for Loop
- Conditional Statements (`if`, `elif`)
- String Methods (`isupper()`, `islower()`)

## ▶️ How to Run

```bash
python count_letters.py
```

## 💻 Sample Output

```
Enter a string: Hello World

Result
Uppercase Letters : 2
Lowercase Letters : 8
```

This project helped me understand:
- How to iterate through a string
- How to use `isupper()` and `islower()`
- How to count characters using loops and conditions
